## If you've never coded before, or have very little experience

If you've never coded before, you'll wanna hit the ground... at least jogging. But hey, there's a start to everything!

Start CodeARobot (you'll need to sign up). Complete Basic Java first, and make sure to take your time.
Then, work on the others, up until Getting Ready for FTC. (Stop right before that, then move on to [Section: Getting Started With FTC](#getting-started-with-ftc).

If you'll be using Android Studio, set it up by following this link:
https://ftc-docs.firstinspires.org/en/latest/programming_resources/android_studio_java/Android-Studio-Tutorial.html

## Kotlin

Kotlin is a language that is pretty much a modern version of Java. It compiles down to JVM, just like Java, so it *can* legally and practically be used in FTC projects.
I personally like Kotlin because it has more language features, and it has null safety.
You should only use Kotlin if you know what you're doing.

To set up Kotlin: https://ftc-docs.firstinspires.org/en/latest/programming_resources/shared/installing_kotlin/Installing-Kotlin.html
