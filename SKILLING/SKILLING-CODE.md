# 🚀 Programming Track
**Languages:** Java (primary), Kotlin (optional), Android Studio  
**Pathing System:** PedroPathing

---

## 🟢 Beginner

### 🎯 Goals
- Learn the basics of Java
- Understand the FTC control system
- Write simple OpModes

### 📚 Training Module
- **Code-A-Robot**  
  [https://www.codearobot.org/](https://www.codearobot.org/)
  - **Modules:**
    - Basic Java
    - Object-Oriented Java
    - Methods
    - Logic & Iteration
    - Arrays and Lists
    - Getting Ready for FTC

### 🧪 Try it out
- **FTCSIM**  
  [https://ftcsim.org/](https://ftcsim.org/)
  - Complete **FTC Intro** and **FTC Movement** challenges.

### 📚 Reference Sites
- **Learn Java For FTC Ebook**  
  [Download PDF](https://github.com/alan412/LearnJavaForFTC/blob/master/LearnJavaForFTC.pdf)
- **FTC Docs – Programming Basics**  
  [https://ftc-docs.firstinspires.org](https://ftc-docs.firstinspires.org)
- **REV Robotics Control System Guide**  
  [https://docs.revrobotics.com/](https://docs.revrobotics.com/)
- **Android Studio Setup for FTC**  
  [Watch Video](https://www.youtube.com/watch?v=Y4dJf7v2y0E)
